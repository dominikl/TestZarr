# TestZarr

A Java command-line application for Zarr operations using JZarr library.

## Requirements

- Java 11 or higher
- Gradle

## Building

```bash
./gradlew build
```

## Running

```bash
./gradlew run --args="--help"
```

Or after building:

```bash
java -jar build/libs/TestZarr-1.0-SNAPSHOT.jar --help
```
